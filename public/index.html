Detalles de Trendy-Tech

TrendyTech es un e-commerce de productos de computación.

Incluye registro, confirmación, autenticación, reseteo de contraseñas, autenticación de terceros con Google, pasarela de pagos con Mercado Pago, Dashboard de admin, borrado lógico, cookies. Como administrador puedes crear, editar y eliminar las propiedades de cada producto en venta, banear y restringir a los usuarios que atenten con las políticas normativas, ver el flujo de ventas. Como usuario debes registrarte o logearte con Google para poder concretar una compra, puedes añadir productos a "favoritos", puedes puntuar tu compra y editar tu perfil.

Tecnologías aplicadas: JavaScript, React, CSS, NodeJS, Express, Sequelize, PostgreSQL, Nodemailer, Mercado Pago, Cloudinary.

Detalles de Henry Food

Single page application que consume la API “FoodApi”. Permite la visualización de la información de las recetas de comida, ver a detalle cada una, filtrarlas, ordenarlas, crear nuevas y buscarlas por su nombre.

Tecnologías aplicadas: JavaScript, React, Redux, HTML, CSS, PostgreSQL, Sequelize, Express, NodeJS.
